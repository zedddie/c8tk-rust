## Usage


```
cargo build --release
./target/release/c8tk_rust program.ch8
./target/release/c8tk_rust --debug program.ch8
```

## Keybindings:

CHIP-8 uses 16 keys. Mapped as:

```
CHIP-8    Keyboard
1 2 3 C   1 2 3 4
4 5 6 D   Q W E R
7 8 9 E   A S D F
A 0 B F   Z X C V
```
